  • 禁止长按链接与图片弹出菜单

a,img { -webkit-touch-callout: none }

  • 禁止ios和android用户选中文字

html,body {-webkit-user-select:none; user-select: none; }

  • 改变输入框placeholder的颜色值

::-webkit-input-placeholder {color: #999; }

  • android上去掉语音输入按钮

input::-webkit-input-speech-button {display: none}

  • 禁用input在ios下，输入英文首字母的默认大写

<input autocapitalize="off" autocorrect="off" />

  • iOS惯性滚动

-webkit-overflow-scrolling:touch

  • 去除高亮

-webkit-tap-highlight-color: rgba(0,0,0,0);

  • 消除transition闪屏

.css {-webkit-transform-style: preserve-3d;
-webkit-backface-visibility: hidden;
-webkit-perspective: 1000;
}

  • 竖屏时样式

@media all and (orientation:portrait){ }

  • 横屏时样式

@media all and (orientation:landscape){ }

炫酷的鼠标悬浮

<style>
    .hover {
        position: relative;
        appearance: none;
        background: #f72359;
        padding: 1em 2em;
        border: none;
        color: white;
        font-size: 1.2em;
        cursor: pointer;
        outline: none;
        overflow: hidden;
        border-radius: 100px;
        width: 100%;
        span {
            position: relative;
        }
        &::before {
            --size: 0;
            content: '';
            position: absolute;
            left: var(--x);
            top: var(--y);
            width: var(--size);
            height: var(--size);
            background: radial-gradient(circle closest-side, #4405f7, transparent);
            transform: translate(-50%, -50%);
            transition: width .2s ease, height .2s ease;
        }
        &:hover::before {
            --size: 400px;
        }
    }
</style>

<script>
    document.querySelector('.hover').onmousemove = (e) => {
        const x = e.pageX - e.target.offsetLeft
        const y = e.pageY - e.target.offsetTop
        e.target.style.setProperty('--x', `${ x }px`)
        e.target.style.setProperty('--y', `${ y }px`)
    }
</script>

position:fixed 降级问题

分析:position:fixed 这个属性,如果其父元素中有使用 transform， fixed 的效果会降级为 absolute。 解决: 保持 fixed与absolute效果一致,使fixed的直接父元素,高度和屏幕高度相同

CSS 变量使用

// sass
$fs14:14px;

// less
@fs14: 14px;

// 原生 css
:root {
    --fs12: 12px;
    --blue: #06f;
}
.css {
    background: var(--blue);
    font-size: var(--fs12);
    font-size: \$fs14;
}

p.img(:portrait="item | transImg")
&[portrait="#{$classU}"] {
    background-image: url(path("/data/images/custom/i_#{$classU}.svg"))
}

1px 方案

@mixin min-device-pixel-ratio ($scale2, $scale3) {

    @media screen and (min-device-pixel-ratio: 2),
    (-webkit-min-device-pixel-ratio: 2) {
        transform: $scale2;
    }

    @media screen and (min-device-pixel-ratio: 3),
    (-webkit-min-device-pixel-ratio: 3) {
        transform: $scale3;
    }
}

@mixin border-1px ($color: #DDD, $radius: 2px, $style: solid) {
    &:before {
        content: "";
        pointer-events: none;
        display: block;
        position: absolute;
        left: 0;
        top: 0;
        transform-origin: 0 0;
        border: 1px $style $color;
        border-radius: $radius;
        box-sizing: border-box;
        width: 100%;
        height: 100%;

        @media screen and (min-device-pixel-ratio: 2),
        (-webkit-min-device-pixel-ratio: 2) {
            width: 200%;
            height: 200%;
            border-radius: $radius * 2;
            transform: scale (.5);
        }

        @media screen and (min-device-pixel-ratio: 3),
        (-webkit-min-device-pixel-ratio: 3) {
            width: 300%;
            height: 300%;
            border-radius: $radius * 3;
            transform: scale (.33);
        }
    }
}

@mixin border-top-1px ($color : #DDD, $style: solid) {
    &:before {
        content: "";
        position: absolute;
        left: 0;
        top: 0;
        width: 100%;
        border-top: 1px $style $color;
        transform-origin: 0 0;
        @include min-device-pixel-ratio (scaleY(.5), scaleY(.33));
    }
}

内联首屏关键 CSS

首次有效绘制（FMP），即指页面的首要内容（primary content）出现在屏幕上的时间. 这一指标影响用户看到页面前所需等待的时间，而 内联首屏关键 CSS（即 Critical CSS，可以称之为首屏关键 CSS）

文字两端对齐

// html
.text-justify
    p 优量通数据
    p 优量通
    p 数据

// css
.text-justify {
    p {
        width: 100px;
        margin: 10px 0;
        border: 1px solid #000;
        text-align: justify;
        text-align-last: justify;

        &:after {
            content: "";
            width: 100%;
        }
    }

}

// JS
orientationchange

    // 竖屏
}

@media screen and (orientation: landscape) {
    // 横屏
}

自动生成段落序号

        p {
            position: relative;
            padding-left: 10px;

            &:before {
            position: absolute;
            left: 0;
            top: 0;
            counter-increment: number;
            content: counter(number) ".";
            margin-right: 2px;
        }
    }
}
